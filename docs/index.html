What is this?

A bunch of data, visualized using text, numbers and lists. At the moment, there is only one data source to choose from and that's my “I watched a Movie!” website.

All of the information is derived from movies I've watched / rewatched. Not very extensive but I believe a fair representation of my film-viewing habit.

The data is separated into cards based on a common theme. Basic numbers info, list cards — one-sentence overview followed by a top 10 (max) list.

For technical details, including data extraction and processing methods, visit the GitHub repo.
